# -Cisco-VPN-Labs-Collection
This repository contains multiple Cisco VPN lab configurations built in Packet Tracer and GNS3. Each lab demonstrates different VPN technologies: 
- Site-to-Site IPsec VPN
- GRE over IPsec VPN
- DMVPN (Dynamic Multipoint VPN)

# -Labs Included
1. ipsec-site-to-site → Basic IPsec VPN between two sites
2. gre-over-ipsec → GRE tunnel secured with IPsec
3. dmvpn-phase1 → DMVPN Phase 1 configuration
